# MIZOKI3 Final Production Site

This is the final packaged MIZOKI3 production website.

## What is included

- Multi-page React/Vite site
- Tailwind CSS styling
- Framer Motion animation
- Lucide icon system
- Production-ready page structure
- Interactive MIZOKI3 Simulation Console
- SRPVDAL Engine monitor
- Decision Control Plane authorization UI
- TCKG / Enterprise Nervous System visuals
- One Brain, Many Lenses domain-cell interface
- Governance, Replay, KPI, Blog, and Contact pages

## Pages

- `/` — full homepage / platform overview
- `/platform` — architecture and anti-dashboard thesis
- `/simulator` — interactive autonomous decision simulator
- `/engine` — SRPVDAL engine deep dive
- `/control-plane` — Decision Control Plane and threshold routing
- `/divisions` — domain cells and divisions
- `/governance` — guardrails, audit, and explainability
- `/kpis` — operational KPI system
- `/blog` — strategic article
- `/contact` — executive demo form shell

## Quick start

```bash
npm install
npm run dev
```

Then open the local Vite URL shown in the terminal.

## Production build

```bash
npm run build
npm run preview
```
